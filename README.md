# 3D Model Generation from Image/Text

## Overview
This prototype converts images or text prompts into 3D models using Shap-E and rembg.

## Dependencies
- Python 3.8+
- See `requirements.txt`

python3 -m venv .venv
.venv\Scripts\activate
pip install -r req.txt
